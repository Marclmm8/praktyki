# Instalacja Linux Mint

## Cel
Przygotowanie komputera do pracy w trakcie praktyk oraz nauka podstaw instalacji i konfiguracji systemu Linux.

## Zakres zadania
- przygotowanie nośnika instalacyjnego,
- uruchomienie instalatora,
- konfiguracja podstawowych ustawień,
- utworzenie użytkownika,
- wykonanie aktualizacji po instalacji.

## Dowody
- screenshot instalatora,
- screenshot pulpitu po instalacji,
- krótki opis przebiegu instalacji.

## Wnioski
To miejsce będzie uzupełniane po wykonaniu zadania.
