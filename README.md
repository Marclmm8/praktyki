# Praktyki myOdoo.pl 2026

Repozytorium służy jako wirtualny dzienniczek praktyk zawodowych dla ucznia technikum informatycznego. Zawiera plan praktyk, wpisy dzienne, artefakty z wykonanych zadań oraz materiały do nauki.

## Strona projektu
Po uruchomieniu GitHub Pages strona będzie dostępna tutaj:

`https://AKTUUALNY_LOGIN.github.io/NAZWA_REPO/`

## Co jest w repo

- `index.html` — strona główna / dashboard praktyk
- `dziennik/` — wpisy dzienne i tygodniowe
- `artefakty/` — większe zadania i efekty pracy
- `materialy/` — linki, checklisty i instrukcje
- `assets/` — screenshoty i grafiki

## Założenia praktyk

- praktyki odbywają się głównie zdalnie,
- uczeń pracuje na sprzęcie przekazanym do domu,
- każde zadanie zostawia prosty ślad: plik, commit, screen albo opis,
- zadania mają być łatwe do sprawdzania i korygowania,
- nie korzystamy z danych produkcyjnych ani nie ingerujemy w infrastrukturę firmową bez potrzeby.

## Moduły praktyk

1. Linux i sprzęt
2. Git i GitHub
3. Odoo — podstawy użytkowe
4. Odoo.sh — sandbox i dev flow
5. Dokumentowanie pracy
6. Mini-projekt końcowy
